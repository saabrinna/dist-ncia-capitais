### Descrição
O projeto consiste na composição de uma tabela que conta com capitais brasileiras e a distancia entre essas 
capitais. As capitais constantes nesse projeto são:

* FORTALEZA
* GOIÂNIA
* JOÃO PESSOA
* MACAPÁ
* MACEIÓ
* MANAUS
* NATAL
* PALMAS
